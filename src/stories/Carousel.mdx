# useCarousel

Provides the behavior and accessibility implementation for a carousel component. A carousel shows a subset of a collection of items, and affords users to scroll through them.

## Usage

```typescript
import {
  useCarousel,
  useCarouselItem,
  useCarouselNavItem,
} from "@rogin/carousel";
```
